This is a UI project for a private dashboard. It's uses node LTS.

Tech stack:
- node@24
- npm
- TypeScript with the most strict setup for the frontend
- TailwindCSS
- Astro package
- Nano Stores for state management
- ky for fetch based HTTP requests
